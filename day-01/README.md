####  what Linux is and it's benefits
It is an Operating System built on or inpired by Unix. It is a free, open-source, customizable and highly secure operating system. It supports a wide range of hardware platforms and widely used in servers, security, and cloud computing

#### Linux distribution and their use cases
A Linux distribution is an operating system built on the Linux kernel, bundled with essential software, tools, and package managers, customized to serve different users like developers, enterprises, cybersecurity professionals, and general users. There are about 600+ Linux distributions. Few popular ones are Ubuntu, Fedora, Kali Linux, Debian, Linux Mint. 

Ubuntu is known for been beginner friendly and mostly used for general use. Debian is the best choice when long term stability and security are prioritized for hosting, cloud setups and corporate systems. For Ethical hacking and cybersecurity use cases Kali Linux is recommended.

####  The Linux architecture

Applications  | System Utilities > Shell > Kernel > Hardware

- Application(System libaraies) - system Libraries provide predefined functions that allow application programs and system utilities to access kernel features without interacting with the kernel directly.

- Kernel - It is the core component of the operating system, that directly interacts with the hardware and manages the hardware resources. It is responsible for memory management, resource allocation, process management, device management, and smooth communication between software and hardware. 

- Shell - Shell is a command interpreter, it takes commands from the user and interprets it to the operating system (kernel), the OS then processes or executes it. examples of shell are Bash, zsh, fish e.t.c.

- Hardware - This is the lowest level of Linux architecture. It consists of physical components that execute commands and provide system resources. The physical components are CPU, RAM, storage, and input/output devices.

- System Utilities- They are built-in (Command-line) tools that help users manage, configure, and maintain the operating system. They simplify system administration by providing ready-to-use commands for common tasks like (file and directory management operations, Monitor system performance and resource usage, Manage users, groups, and permissions, Configure and troubleshoot network settings)

####  Difference Between Terminal , Shell and Command Line interface

While these three are often used interchangeably as the word "CLI",  they do and mean different things in true sense.

- Terminal is a visual environment that allows users to interact with the operating system using commands. It is just an interface that accepts user input and displays output

- Shell Interpretes the commands to the Operating system for it to be processed

- The Command line Interface is a broad term it includes the command, terminal, and shell. It is a method of interacting with a computer by typing commands, which are interpreted by a shell and displayed in a terminal.
